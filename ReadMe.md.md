## banco de dados
Uma estrutura de dados que agrupa-os em tabelas. estas tabelas, representam um conjunto de entidades.
ex do nosso banco de dados... precisamos guardar dados de acesso dos usuários do sistemas(estoquista,admin,vendedor). guardar dados do produto, imagens, indicações, contra-indicações, descrições, validade,vendas e itens de venda, etc.

estes dados são guardados em colunas!
estes dados têm tipos(que basicamente são formatos dos mesmos). ex. uma data é composta de numeros reprensentando dia, mês e ano e ou hora.
